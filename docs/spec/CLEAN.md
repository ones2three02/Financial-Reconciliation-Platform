# 数据清洗规范 (CLEAN.md)

## 数据清洗规则

1. **金额标准化**：
   * 所有金额必须转换为标准 Python `Decimal` 格式。
   * 必须剔除货币符号（`¥`, `$`, `元`）、千分位逗号（`,`）。
   * 必须正确解析负数的会计表示（例如 `(100.00)` 转换为 `-100.00`）。
   
2. **日期标准化**：
   * 所有交易日期统一转换为 Python `datetime.date` 对象（持久化为 `YYYY-MM-DD`）。
   * 必须支持 `/`、`-`、无分割符等常见日期格式的自动推导，并处理 Excel 内置的 Serial Number 数字日期偏移。
   
3. **空值与脏数据**：
   * 任何一行的核心字段（日期、金额、原始门店名）缺失或格式严重错误，该行标记为 `is_valid = False`，清洗状态记为 `error`，并记录具体错误到 `error_message`。
   * 绝对不允许在清洗阶段直接物理删除脏数据行，必须完整保留该记录用于界面纠错。
