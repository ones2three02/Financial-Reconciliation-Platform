# 数据解析规范 (PARSER.md)

## 自动识别与解析规则
1. **自动列名嗅探**：解析引擎读取 Excel 头部时，应首先检测数据库 `field_mapping` 中该渠道配置的列名。如果不存在映射，则采用内置的模糊匹配关键字（如“实收”、“金额(元)”、“交易净额” ➜ 映射为 `amount`）。
2. **Sheet 默认处理**：若 Excel 包含多个 Sheet，默认只解析第一个活动工作表（Sheet 0）。
3. **元信息保留**：解析后的每一行，都必须将本次解析所采用的列头映射规则（`_detected_mappings`）嵌入到 raw_data 实体中，以便数据清洗层能够精准按键提取。
