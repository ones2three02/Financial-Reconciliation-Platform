# 财务自动对账平台 (Financial Reconciliation Platform)

FRP 是专为多门店、多渠道交易设计的企业级财务自动对账系统。

## 项目背景
当前集团拥有约 22 家门店，财务每天需要人工核对所有门店的销售收入。数据来源包含通联好老板、美团、抖音、手工现金账和收银系统销售数据。由于各三方渠道导出 Excel 模板不一、门店名称不统一（例如“杨一一游泳馆”与“杨一一店”为同一家店）、数据量庞大（每日几十个 Excel 文件），传统的人工核对流程费时费力且极易出错。

FRP 平台旨在通过自动化的文件导入、字段映射解析、数据清洗、门店标准化以及智能对账引擎，将上述繁重的工作缩减为“一键对账”，并提供直观的异常追踪看板。
